## Description
MariaDB in docker compose with phpmyadmin.

Tag: Apple M1 chip compatible MySql

## Installation

1. Clone the repo.
2. `cd <repo-directory>`
3. `docker compose up -d`

## Access phpMyAdmin

http://localhost:8085

User: `root`

Password: `MairaFala`

## Access mysql DB server

`mysql -h 127.0.0.1 -P 3306 -u root -pMairaFala`
